# Zhouyi Divination - 周易占卜 Skill

基于蓍草起卦法（四营十八变）的 Claude Code Skill，模拟传统周易占卜全流程：起卦、纳甲、定世应、联网检索权威解读、输出 Markdown 解卦报告。

## 快速开始

### 安装

```bash
git clone https://github.com/Sheaffer-triumph/ZhouyiDivination
cp -r ZhouyiDivination/zhouyi-divination ~/.claude/skills/
```

### 依赖

脚本依赖 `lunar-python` 获取农历时间。Skill 首次运行时会自动检测并提示安装：

```bash
pip install lunar-python
```

### 使用

在 Claude Code 会话中，输入 `/zhouyi-divination` 直接启动，或说"帮我算一卦"、"算算事业"等，Skill 会自动触发。

启动后按提示操作：
1. 提供你的 Python 解释器路径（如 `python3`）
2. 说出你想占卜的问题（事业、感情、财运等）
3. Skill 自动起卦、联网检索权威资料、完成解读
4. 最终输出一份 Markdown 报告，保存在当前工作目录

## 工作流程

1. **确认 Python 路径** -- 适配任意环境的 Python 解释器，自动检测 `lunar-python` 依赖
2. **获取问事项目** -- 对话式收集用户想占卜的具体问题
3. **蓍草起卦** -- 运行 `scripts/yi.py`，以正态分布模拟随机分堆，执行四营十八变成卦
4. **联网搜索** -- 根据本卦名和动爻位置，搜索《周易》原文及王弼、孔颖达、朱熹、程颐、南怀瑾等易学家注解
5. **结构化解读** -- 严格按四步框架分析：寻用神、世应生克、原典引述、宏观推演
6. **输出报告** -- 生成包含原始卦象和完整解读的 Markdown 文件

## 输出示例

每次占卜生成两个文件：

- `divination_YYYYMMDD_HHMMSS.txt` -- 起卦原始输出（本卦、之卦、互卦、世应、地支五行、爻辞、占卜时间）
- `divination_report_YYYYMMDD_HHMMSS.md` -- 完整解读报告（基本信息、原始卦象、四步解读、参考来源）

## 算法说明

`scripts/yi.py` 实现了传统蓍草起卦的完整算法：

- **分二**：49 根蓍草随机分为左右两堆，使用正态分布模拟物理分堆的随意性
- **挂一**：从右堆取出一根
- **揲四**：左右分别模 4 取余（余 0 视为 4）
- **归奇**：取出的蓍草合并，剩余继续下一轮
- **三变成一爻**：重复三次后，剩余蓍草数除以 4 得到爻值（6=老阴、7=少阳、8=少阴、9=老阳）
- **六爻成卦**：重复六次得本卦，老阴/老阳为动爻，动爻阴变阳、阳变阴得之卦

## 解读框架

解读严格遵循"文王六爻预测法"的结构化框架：

1. **寻用神与环境压力测试** -- 六亲定位，结合日建月建五行判断用神衰旺
2. **世应互动关系** -- 分析世爻（问卜者）与应爻（目标）的五行生克
3. **原典重现与突变因素追踪** -- 一字不漏引述《易经》原文、白话释义、结合现状深度解析
4. **宏观推演与落地建议** -- 本卦到之卦的意象转换，给出具体行动建议

## 卦象查询键编码说明

六十四卦以 6 位二进制字符串索引，从初爻到上爻：阴爻（6/8）为 `0`，阳爻（7/9）为 `1`。

例如：`111101` = 火天大有（初阳、二阳、三阳、四阳、五阴、上阳）
