# 周易占卜

模拟传统蓍草起卦法（四营十八变），生成六爻卦象，结合《易经》原文和联网搜索进行结构化解读。

## 第一步：确认 Python 路径

询问用户可用的 Python 解释器路径。示例：
- `python3`
- `~/software/miniforge3/envs/Isvara/bin/python`
- 用户指定的任意路径

验证该 Python 是否有 `lunar-python` 库：
```bash
<PYTHON_PATH> -c "import lunar_python" 2>&1
```
如果没有安装，提示用户：`pip install lunar-python`

## 第二步：获取问事项目

询问用户想要占卜的具体问题，例如：事业发展、感情婚姻、财运投资、健康状况、学业考试等。

## 第三步：起卦

切换到用户当前工作目录，运行起卦脚本：

```bash
cd <用户工作目录> && <PYTHON_PATH> <SKILL_DIR>/scripts/yi.py
```

将完整原始输出保存为 `divination_YYYYMMDD_HHMMSS.txt`。

## 第四步：联网搜索权威资料

根据本卦名和动爻位置，使用 `mcp__exa__web_search_exa` 或 `WebSearch` 搜索权威易经解读：

1. 搜索 `《易经》<本卦名>卦 卦辞 解读 周易`
2. 如有动爻，搜索 `<本卦名> <动爻位置> 爻辞 解读`
3. 搜索 `<本卦名> 五行 世应 六爻 分析`
4. 搜索 `<本卦名> 之 <之卦名> 卦象 解读`（如有之卦）

获取至少 2 个来源用于交叉参考。优先选取含有《周易》原文引述或知名易学家（如王弼、孔颖达、程颐、朱熹、尚秉和、高亨、南怀瑾等）注解的内容。

## 第五步：结构化解读

严格按以下四步框架解读。详细解读指引见 `references/interpretation-prompt.md`。

### 寻用神与环境压力测试
明确指出用神（六亲角色）。结合占卜时间的日建月建五行属性，白话说清：用神当前是被日月生扶还是克泄。这一部分要让人感觉到"这件事的核心角色是什么"以及"天时大环境对它好不好"。

### 世应互动关系
分析世爻与应爻的五行生克。说清问卜者与目标之间是相向而行还是彼此消耗。用平常人的语言解释"世克应、应克世、世应比和、世生应、应生世"分别意味着什么。

### 原典重现与突变因素追踪
- **【引经据典】** 一字不漏引用《易经》原文（动爻爻辞，无动爻则用本卦卦辞）。
- **【古文释义】** 用现代白话解释原文的字面意思和核心意象。
- **【深度解析】** 结合白话意象，分析变动能量对用神/世爻的具体影响。引用联网搜索结果中的权威观点作为佐证。

### 宏观推演与落地建议
从本卦→之卦的意象转换，给出事件的大结局走向。必须提供有建设性的具体行动建议，拒绝虚词和套话。

## 语气与风格
- 学术、理性、客观——把玄学当数据分析来讲
- 专业术语配合白话语解释，让完全不懂易经的人也能听懂
- 展现国学底蕴，但拒绝"天机不可泄露"式的封建迷信话术
- 给建议时要落地、具体、可执行——不要"顺应天道"这种正确的废话
